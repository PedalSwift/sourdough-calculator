# 🥖 Sourdough Calculator

A simple browser-based sourdough baker’s percentage calculator.

## Features

* Hydration-based calculations
* Starter (100% hydration) built in
* Dough weight or flour-based mode
* Save and reload recipes
* Print/export functionality

## Use it here:

👉 https://PedalSwift.github.io/sourdough-calculator/

## About

Built for experimenting with sourdough recipes and dialing in the perfect loaf.
